- 길이가 5인 배열에서 만약 0~2번째 까지 N만큼 감소시키고 1~3번째 까지 M만큼 감소시키는 경우 아래와 같이 배열을 만듦
  - [-N, -M, 0, N, M, 0]
    - -N : 0, N : 2 + 1 
    - -M : 1, M : 3 + 1
